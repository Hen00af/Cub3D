## How to play
in terminal, execute this command
```sh
  git clone https://github.com/42paris/minilibx-linux.git && (cd ./minilibx-linux && make)
  make
```
and execute this command

```sh
  ./cub3D maps/validmaps/{map_name}.cub
```

then you can play this game.
